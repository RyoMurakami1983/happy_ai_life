# session-share-document naming

## 保存先

- `docs/sessions/YYYYMMDD-HHmmss-Session名.md`

## ルール

- `()` は使わない
- 禁止文字は置換する
- 長すぎるタイトルは短くする
- 読みやすさを優先して、日付 + ハイフン + タイトルで揃える

## commit 件名

- `docs:共有タイトル`
- ふりかえり由来なら、最終決定したタイトルをそのまま使う

## 例

```text
20260410-103000-furikaeri-practice-smart-output.md
```
