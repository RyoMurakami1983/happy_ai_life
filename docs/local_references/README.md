# local_references の運用ルール

`docs/local_references` は、改善や調査のための**個人用・一時的な参考資料置き場**です。
このフォルダの中身は原則として Git 管理しません。

## このフォルダに置くもの

- 調査メモ
- 一時的な比較資料
- 外部ドキュメントの抜粋や参照リンク
- 実装時の参考にする私的な下書き

## 置かないもの

- 機密情報
- 再配布を前提にした完成文書
- 共有すべき正式仕様
- 長期保存したい成果物

## 運用方針

- 中身はローカル運用を優先する
- 共有したい文書は `docs/` 配下の tracked な場所へ昇格させる
- このフォルダのルールだけは、この `README.md` で明文化する

## 補足

- `README.md` は例外として Git 管理する
- `.gitkeep` はディレクトリ維持のために残す
