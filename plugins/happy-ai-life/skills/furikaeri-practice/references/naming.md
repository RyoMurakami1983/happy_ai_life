# furikaeri naming

## 保存先

- `docs/furikaeri/YYYYMMDD-HHmmss-タイトル.md`
- `.copilot/docs/furikaeri/YYYYMMDD-HHmmss-タイトル.md`

## ルール

- repo 用と home 用は同じ basename にする
- `()` は使わない
- 禁止文字は置換する
- 長すぎるタイトルは短くする
- 読みやすさを優先して、日付 + ハイフン + タイトルで揃える

## commit 件名

- `docs:ふりかえりタイトル`
- 共有保存由来なら、最終決定したタイトルをそのまま使う

## 例

```text
20260410-103000-furikaeri-practice-dialog-and-storage.md
```
